## 📝 使用指南

### 作为开发者
1. 查看 [任务追踪表](tracking/TASK_TRACKING.md) 了解当前任务
2. 阅读 [项目规划](../PROJECT_REFACTOR_PLAN.md) 理解整体规划
3. 使用 [每日日志模板](templates/DAILY_LOG_TEMPLATE.md) 记录工作
4. 更新任务状态到任务追踪表

### 作为项目经理
1. 查看 [任务追踪表](tracking/TASK_TRACKING.md) 了解进度
2. 查看 [风险管理文档](RISK_MANAGEMENT.md) 了解风险
3. 审查每日工作日志
4. 更新项目主规划

### 作为AI助手
1. 阅读 [项目规划](../PROJECT_REFACTOR_PLAN.md) 理解目标
2. 查看当前进行中的任务
3. 按照任务分解执行工作
4. 使用模板记录工作
