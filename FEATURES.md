# Pi Gateway 功能规格

> **产品定位**：Pi Coding Agent 的 Web 网关，为开发者提供美观的 AI 聊天界面和文件管理功能。
> 
> **核心价值**：在浏览器中实现与 AI 助手的自然对话，同时管理本地文件系统。

---

## 1. 界面架构

7. 完成后可继续对话

### 2.4 FileBrowser（文件浏览器）

#### 布局位置
- **主内容区**，与 ChatPanel 互斥显示
- **左侧**：文件目录树面板（可折叠）
- **右侧**：文件列表/编辑器区域
- **底部**：可弹出面板（在固定菜单上方）

#### 文件目录树（左侧面板）
- **异步加载**：点击展开目录时异步加载子目录和文件
- **懒加载**：初始只加载根目录，逐级展开时加载
- **状态指示**：加载中显示 spinner，加载完成显示内容
- **缓存机制**：已加载的目录缓存，避免重复请求

#### 底部面板系统
- **位置**：在固定底部菜单上方弹出
- **层级**：可覆盖底部菜单，但低于主内容区
- **功能**：
  - 显示文件执行结果
  - 显示命令输出
  - 显示文件预览
- **控制**：可调整高度、最小化、关闭

#### 功能清单

| 功能 | 状态 | 用户价值 | 操作路径 |
|------|------|----------|----------|
| **目录树浏览** | ✅ | 可视化文件结构 | 左侧面板点击展开目录 |
| **异步加载** | ✅ | 快速初始加载 | 点击目录时加载子项 |
| **文件排序** | ✅ | 多维度排序 | 点击表头（时间/名称/类型/大小） |
| **文件过滤** | ✅ | 快速查找文件 | 输入框输入关键词 |
| **文件预览** | ✅ | 查看文件内容 | 点击文件，支持代码/图片/HTML |
| **文件编辑** | ✅ | 修改文件内容 | 预览模式下点击编辑 |
| **文件保存** | ✅ | 保存修改 | 编辑器内 Ctrl+S |
| **文件执行** | ✅ | 执行脚本文件 | 选中 .sh/.py/.js 点击执行 |
| **批量选择** | ✅ | 多文件操作 | 勾选框多选 |
| **底部面板** | ✅ | 查看执行结果 | 执行文件后自动弹出 |

#### 交互流程：文件操作

##### 1. 目录操作
1. 点击目录树中的目录
2. 异步加载子目录和文件
3. 右侧文件列表更新显示该目录内容
4. 面包屑导航更新

##### 2. 文件选择与查看
1. **单击文件**：选中状态（高亮显示）
2. **再次单击**：打开主导排查（主内容区预览）
3. **双击文件**：直接打开主导排查

##### 3. 文件编辑
1. 在主导排查中点击"编辑"按钮
2. 打开编辑窗口（模态框或新面板）
3. 修改文件内容
4. Ctrl+S 保存（或点击保存按钮）
5. 显示保存成功提示
6. 关闭编辑器返回文件列表

##### 4. 文件执行
1. 选中可执行文件（.sh, .py, .js, .ts等）
2. 点击"执行"按钮或右键菜单选择"执行"
3. **服务器端执行**：命令在服务器上执行
4. **底部面板弹出**：自动弹出底部面板
5. **实时输出**：显示执行进度和输出结果
6. **执行完成**：显示最终结果和退出码

#### 技术实现

##### 异步目录加载
```typescript
// 伪代码示例
async function loadDirectory(path: string) {
  showLoading(path);
  try {
    const { directories, files } = await api.files.list(path);
    updateDirectoryTree(path, directories, files);
  } catch (error) {
    showError(`加载目录失败: ${path}`);
  } finally {
    hideLoading(path);
  }
}
```

##### 底部面板管理
```typescript
// 伪代码示例
class BottomPanelManager {
  open(type: 'execution' | 'preview', content: any) {
    // 弹出底部面板
    // 设置内容
    // 调整高度
  }
  
  close() {
    // 关闭面板
  }
  
  minimize() {
    // 最小化面板
  }
}
```

##### 服务器端执行
```typescript
// 伪代码示例
async function executeFile(filePath: string) {
  const result = await api.files.execute(filePath, {
    cwd: getCurrentDirectory(),
    env: process.env,
  });
  
  // 流式输出到底部面板
  bottomPanel.open('execution', {
    filePath,
    output: result.output,
    exitCode: result.exitCode,
    duration: result.duration,
  });
}
```

#### 用户体验优化
- **加载状态**：目录加载时显示 spinner
- **错误处理**：加载失败显示错误信息，可重试
- **键盘导航**：支持键盘上下左右导航文件树
- **快捷键**：
  - `Enter`：打开选中文件
  - `F5`：刷新当前目录
  - `Ctrl+F`：聚焦搜索框
  - `Esc`：关闭底部面板
- **响应式设计**：移动端适配，目录树可折叠
