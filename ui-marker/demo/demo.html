React UI Marker v2.0

一个浏览器端可视化选择器，帮助 AI 理解页面元素

✨ 如何使用

1. 激活

按 Ctrl+Shift+M 激活标记系统

2. 选择

鼠标悬停查看元素信息，点击选择目标元素

3. 描述

在弹出的对话框中描述修改需求，发送给 AI

💡 提示：此页面已加载 UI Marker，按 Ctrl+Shift+M 即可开始体验！

🔧 技术说明

三层架构

  • ✅ Level 1: DOM API，100% 稳定，零依赖
  • ✅ Level 2: data-* 标记，增强精度
  • ✅ Level 3: React 信息，机会性获取

输出信息

  • 📄 DOM 路径和属性
  • 🔍 文本内容和上下文
  • 🎯 高置信度标识符
  • 🗺️ AI 搜索策略建议

开发者选项

// 自定义配置
const marker = new UIMarker({
  onSubmit: (payload) => {
    console.log('用户选择:', payload);
    // 发送给 AI 处理...
  },
  shortcut: 'Ctrl+Shift+M'
});

// 手动控制
marker.activate();
marker.deactivate();
